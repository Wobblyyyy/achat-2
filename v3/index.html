welcome to a_ch_at

about

a_ch_at is a browser-based chat service. users can create group chats, message other users in direct messages, chat on the global server, play games with eachother, earn tokens by using the app to buy cool cosmetics, and more.

group chats

users can create group chats with other members, with no restrictions on how many users can be in the group. every group features many of the same things everything else, including DMs has. you can still earn tokens to spend on epic stuff, and group messages are saved up to 100 messages back.

direct messages

direct messages are exactly what the name would suggest - a way to talk to one singular person. by default, messages in a direct message log are saved up to 50 messages back, but that can be upgraded to 100 or even 200 with the use of tokens.

tokens

tokens are earned through very mysterious ways. you can pay other users tokens, use tokens to buy cosmetics, or use tokens to upgrade whichever chats you'd like.

shop

a_ch_at's shop allows you to spend your hard-earned tokens on a variety of things. users can buy the ability to use different name colors or chat colors in direct messaging groups.

directive

a_ch_at is designed to be a fast, free browser based chat solution with a variety of features just to make it a little bit less boring than your standard chat service. a_ch_at is written entirely in HTML, CSS, and JS and is designed to work on Chromium based web browsers - if you're having difficulty you should switch to one of the following web browsers
- Google Chrome
- Opera
- Vivaldi
- Brave
- Microsoft Edge Beta (note that regular Microsoft Edge may not work)

development

a_ch_at is not currently designed to be open source, although will convert to this model in the near future.

javascript

everything on the web client for a_ch_at is written in javascript. with the latest re-design, we've switched off from using jquery to the much smaller and similar library score for the purpose of decreasing load time and bandwidth usage. we also rely on several javascript libraries, such as
- jscookie
- sweetalert2
- firebase
